1. What is a conditional statement? Give three examples.
A conditional statement is a check on whether is statement, or statements, is true or false. A few examples:
'''ruby
if x == true
    total++
end 

while y > 0
    y--
end

case z
when 1
    puts "z is one!:
when 2
    puts "z is two!"
else
    puts "I don't know what it is."
end
'''
